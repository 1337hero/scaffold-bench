---
title: SVG Self-Portrait
category: svg-art
---

Write a single self-contained HTML file (no external dependencies, no frameworks, no CDN) that displays a creative stylized self-portrait drawn entirely with SVG elements. Requirements:

- Pure HTML + inline CSS + inline SVG only.
- The SVG should fill a substantial portion of the viewport.
- Use geometric shapes, paths, gradients, and patterns to create an abstract/expressive portrait.
- Include at least: eyes, mouth, and head shape using SVG primitives and paths.
- Use a cohesive color palette with gradients.
- Add subtle CSS animations (e.g., blinking eyes, slight head tilt, color shift).
- Include a small caption below the portrait.
- The portrait should feel artistic and intentional, not just stick figures.
