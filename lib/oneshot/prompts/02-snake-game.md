---
title: Snake Game
category: game
---

Write a single self-contained HTML file (no external dependencies, no frameworks, no build step, no CDN) that implements the classic Snake game. 

Requirements:

- Pure vanilla JS + HTML + CSS only, inline in one file.
- Use an HTML5 `<canvas>` for rendering the game.
- Snake moves continuously; player controls direction with arrow keys.
- Food appears randomly; eating it grows the snake and increases score.
- Game over when snake hits the wall or itself.
- Show score, high score, and a "Press any key to start" message.
- Clean, modern visuals: dark background, bright green snake, red food.
- Grid-based movement with smooth rendering.
